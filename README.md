# TaxiPro - Taxi Management System

A desktop taxi management system developed using C#, Avalonia UI, Entity Framework Core, and SQLite.

## Features

* Admin Login
* Driver Login
* Driver Management
* Customer Management
* Order Management
* Earnings Tracking
* Reviews System
* Withdrawal Requests
* Driver Profile
* Dark Mode

## Technologies

* C#
* .NET
* Avalonia UI
* Entity Framework Core
* SQLite
